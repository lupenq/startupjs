# ArrayInput

ArrayInput lets you build dynamic forms using its declarative api.

Possible types are:

[—Åheckbox](/docs/forms/Checkbox)
[text](/docs/forms/TextInput)
[object](/docs/forms/ObjectInput)
[select](/docs/forms/Select)
[date](/docs/forms/DateTimePicker)
[datetime](/docs/forms/DateTimePicker)
[time](/docs/forms/DateTimePicker)

The `items` object also inherits any of the props passed to these components.

```jsx
import { ArrayInput } from '@startupjs/ui'
```

## Simple example

```jsx example
const [, $value] = useValue(['Green', 'Blue'])
return (
  <ArrayInput
    $value={$value}
    label='Colors'
    items={{
      input: 'text'
    }}
  />
)
```

## Properties
- `style` - styles for the main `ArrayInput` container,
- `inputStyle` - styles for the input container,
- `$value` - model with fields for `ArrayInput`,
- `label` - displayed title,
- `items` - an object with a required key `input` to indicate what type of input to display.
