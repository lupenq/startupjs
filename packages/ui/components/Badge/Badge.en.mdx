# Badge

Badge is small status descriptors for UI elements.

```js
import { Badge } from '@startupjs/ui'
```

## Simple exmaple

```jsx example
return (
  <Badge>
    <Div styleName='child'/>
  </Badge>
)
```

## Sizes

Badges come in three different sizes: `s`, `m`, and `l`. The `m` size is the default and most frequently used. The other sizes should be used sparingly to create a hierarchy of importance on a page.

```jsx example
return (
  <Row>
    <Badge label='16' size='s'>
      <Div styleName='child' pushed/>
    </Badge>
    <Badge label='20'>
      <Div styleName='child' pushed/>
    </Badge>
    <Badge label='24' size='l'>
      <Div styleName='child' pushed/>
    </Badge>
  </Row>
)
```

## Variants

Badges are of two types: `default` and `dot`. `default` it is the most frequently used and the default specified. `dot` turns the icon into a small dot, this can be used as a message that something has changed without content. If the property is `variant='dot'`, the `size` property will be ignored.

```jsx example
return (
  <Row>
    <Badge label='10'>
      <Div styleName='child' pushed/>
    </Badge>
    <Badge variant='dot'>
      <Div styleName='child' pushed/>
    </Badge>
  </Row>
)
```

## Colors

Color is `primary` by default. It can be changed by passing color name from the config colors to `color` property. Possible values of property can be found in the `Sandbox` section at the bottom of the page.

```jsx example
return (
  <Badge color='error'>
    <Div styleName='child' pushed/>
  </Badge>
)
```

## Position

Badges come in two positions: `top` and `bottom`. Default value is `top`.

```jsx example
return (
  <Row>
    <Badge>
      <Div styleName='child' pushed/>
    </Badge>
    <Badge position='bottom'>
      <Div styleName='child' pushed/>
    </Badge>
  </Row>
)
```

## Icons

Icon can be added using `icon` property.

```jsx example
return (
  <Badge icon={faThumbsUp} label='10+' color='success'>
    <Div styleName='child' pushed/>
  </Badge>
)
```

## Counter

If `label` property takes a value of type `number`, then the `max` property can be set to limit the maximum displayed number in the label.

```jsx example
return (
  <Badge label={1000} max={100}>
    <Div styleName='child' pushed/>
  </Badge>
)
```

## renderLabel

The `renderLabel` property takes a render function and renders it instead of inside the Badge.

```jsx example
const renderCustomLabel = () => (
  <Button
    onPress={() => console.log('Clicked')}
    size='xs'
    variant='text'
    color='white'
  >
    Click me
  </Button>
)

return (
  <Badge renderLabel={renderCustomLabel} size='l'>
    <Div styleName='child' pushed/>
  </Badge>
)
```
