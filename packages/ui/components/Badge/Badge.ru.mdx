# Badge (Бейдж)

Бейджи - это небольшие дескрипторы статуса для элементов пользовательского интерфейса.

```js
import { Badge } from '@startupjs/ui'
```

## Простой пример

```jsx example
return (
  <Badge>
    <Div styleName='child'/>
  </Badge>
)
```

## Sizes

Бейджи бывают трех разных размеров: `s`, `m` и `l`. Размер `m` является наиболее часто используемым и определен по умолчанию. Остальные размеры следует использовать с осторожностью, чтобы создать иерархию важности на странице.

```jsx example
return (
  <Row>
    <Badge label='16' size='s'>
      <Div styleName='child' pushed/>
    </Badge>
    <Badge label='20'>
      <Div styleName='child' pushed/>
    </Badge>
    <Badge label='24' size='l'>
      <Div styleName='child' pushed/>
    </Badge>
  </Row>
)
```

## Variants

Бейджи бывают двух видов: `default` и `dot`. `default` является наиболее часто используемым и определен по умолчанию. `dot` превращает значок в маленькую точку, это можно использовать как уведомление о том, что что-то изменилось без контента. При значении свойства `variant='dot'` свойство `size` будет игнорироваться.

```jsx example
return (
  <Row>
    <Badge label='10'>
      <Div styleName='child' pushed/>
    </Badge>
    <Badge variant='dot'>
      <Div styleName='child' pushed/>
    </Badge>
  </Row>
)
```

## Colors

По умолчанию цвет является `primary`. Его можно изменить, передав имя цвета из цветов конфигурации в свойство `color`. Возможные значения свойства можно найти в песочнице внизу страницы.

```jsx example
return (
  <Badge color='error'>
    <Div styleName='child' pushed/>
  </Badge>
)
```

## Positions

Бейджи бывают в двух позиций: `top` и `bottom`. Значение по умолчанию - `top`.

```jsx example
return (
  <Row>
    <Badge>
      <Div styleName='child' pushed/>
    </Badge>
    <Badge position='bottom'>
      <Div styleName='child' pushed/>
    </Badge>
  </Row>
)
```

## Icons

Иконка может быть добавлена с помощью свойства `icon`.

```jsx example
return (
  <Badge icon={faThumbsUp} label='10+' color='success'>
    <Div styleName='child' pushed/>
  </Badge>
)
```

## Счетчик

Если свойство `label` принимает значение типа `number`, то можно задать свойство `max` чтобы ограничить максимальное отображаемое число в лейбле.

```jsx example
return (
  <Badge label={1000} max={100}>
    <Div styleName='child' pushed/>
  </Badge>
)
```

## renderLabel

Свойство `renderLabel` принимает рендер-функцию и рендерит её вместо внутри бейджа.

```jsx example
const renderCustomLabel = () => (
  <Button
    onPress={() => console.log('Clicked')}
    size='xs'
    variant='text'
    color='white'
  >
    Click me
  </Button>
)

return (
  <Badge renderLabel={renderCustomLabel} size='l'>
    <Div styleName='child' pushed/>
  </Badge>
)
```
