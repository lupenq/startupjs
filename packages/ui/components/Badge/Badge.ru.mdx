# Badge (Бейдж)

Бейджи - это небольшие дескрипторы статуса для элементов пользовательского интерфейса. В них должен быть контент для доступности и ясного понимания. Если контент не определен, бейдж становится только кругом.

```js
import { Badge } from '@startupjs/ui'
```

## Простой пример

```jsx example
return (
  <Badge>
    <Div styleName='child'/>
  </Badge>
)
```

## Variants

Бейджи бывают двух видов: `default` и `dot`. `default` является наиболее часто используемым и определен по умолчанию. `dot` превращает значок в маленькую точку, это можно использовать как уведомление о том, что что-то изменилось без контента.

```jsx example
return (
  <Row>
    <Badge label='10'>
      <Div styleName='child' pushed/>
    </Badge>
    <Badge variant='dot'>
      <Div styleName='child' pushed/>
    </Badge>
  </Row>
)
```

## Colors

По умолчанию цвет является `primary`. Его можно изменить, передав имя цвета из цветов конфигурации в свойство `color`. Возможные значения свойства можно найти в песочнице внизу страницы.

```jsx example
return (
  <Badge color='error'>
    <Div styleName='child' pushed/>
  </Badge>
)
```

## Positions

Бейджи бывают в двух позиций: `top` и `bottom`. Значение по умолчанию - `top`.

```jsx example
return (
  <Row>
    <Badge>
      <Div styleName='child' pushed/>
    </Badge>
    <Badge position='bottom'>
      <Div styleName='child' pushed/>
    </Badge>
  </Row>
)
```

## Icons

Иконка может быть добавлена с помощью свойства `icon`.

```jsx example
return (
  <Badge icon={faThumbsUp} label='10+' color='success'>
    <Div styleName='child' pushed/>
  </Badge>
)
```
