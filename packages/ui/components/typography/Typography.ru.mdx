# Компоненты типографии

Наследует [Text props](https://reactnative.dev/docs/next/text).

Набор компонентов, чтобы сделать текст понятным и легко читаемым для конечного пользователя.

```jsx
import { Span, H1, H2, H3, H4, H5, H6 } from '@startupjs/ui'
```

## Span (спан)

Базовый компонент типографики - это `Span`

```jsx example
  <Span>Lorem ipsum dolor sit amet, consectetur adipisicing elit</Span>
```

## Размеры

Для изменения размера текста используйте глобальный миксин `font(size)`. Возможные значения `size` (`h1`, `h2`, `h3`, `h4`, `h5`, `h6`, `body1`, `body2`). `body2` - это стандартный размер `Span`.

В `.styl` файле
```stylus
.text
  font(h2)
```

```jsx example
  <Span styleName='text'>Text size - xxl.</Span>
```

## Заголовки

`H*` комопненты используются для заголовок или подзаголовков раздела или страницы, а также вставляют семантические теги в DOM на вебе.

```jsx example
return (
  <Div>
    <H1>H1 Heading</H1>
    <Br half />
    <H2>H2 Heading</H2>
    <Br half />
    <H3>H3 Heading</H3>
    <Br half />
    <H4>H4 Heading</H4>
    <Br half />
    <H5>H5 Heading</H5>
    <Br half />
    <H6>H6 Heading</H6>
  </Div>
)
```

## Жирный текст

Чтобы сделать текст жирным, передайте компоненту свойство `bold`.

```jsx example
return (
  <Div>
    <Span bold>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</Span>
    <H4 bold>H4 Heading</H4>
  </Div>
)
```

## Курсив

Чтобы сделать текст курсивом, передайте компоненту свойство `italic`.

```jsx example
return (
  <Div>
    <Span italic>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</Span>
    <H4 italic>H4 Heading</H4>
  </Div>
)
```
